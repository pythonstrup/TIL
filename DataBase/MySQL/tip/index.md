## Index는 언제 걸지 말아야 하는가?

### 1. HAVING 절

- `GROUP BY`에 사용된 조건은 인덱스를 사용해 처리될 수 없으므로 `HAVING` 절을 튜닝하려고 인덱스를 생성하거나 다른 방법을 고민하지는 말자.

### 2. 레코드의 양이 적을 때

- 레코드의 양이 적다면, 테이블 스캔이 인덱스 조회보다 오버헤드가 적을 수 있다.
- 데이터를 지속적으로 추가해나갈 테이블이 아니라면 인덱스 사용을 자제하자.
