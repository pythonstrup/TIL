# Shell

## 쉘이란?

- 디스크에는 적으면 수십 개, 많으면 수백 개의 프로그램들이 존재한다. 이 프로그램들이 언제 메모리에 로딩되고 언제 메모리에서 해제될까? 그것을 관리해주는 프로그램이 쉘(Shell)이다.
